# Youtube downloader bot
## About
lib:
  - aiogram
  - yt-dlp

Simple bot for downloading youtube video

## Installation
clone the repo
```sh
git clone https://github.com/ZeroNiki/Youtube-Downloader-Bot.git
cd Youtube-Downloader-Bot
```

## Configuration
go to `src/.env` and insert your token from @BotFather:
```
TOKEN=your token 
```

### Start

```
python3 main.py
```
